# Weather Web App 🌦️

A responsive and user-friendly Weather Web Application built using modern web technologies and integrated with a weather API to provide real-time weather information for any location worldwide.

The application allows users to search for cities and instantly view current weather conditions, including temperature, humidity, wind speed, atmospheric pressure, and overall weather status. By fetching live data from an external API, the app delivers accurate and up-to-date weather forecasts in an intuitive interface.
--> https://temp-reports.netlify.app/

## Features

* 🌍 Search weather information for any city worldwide
* 🌡️ Display real-time temperature and weather conditions
* 💨 View wind speed, humidity, and pressure details
* ☁️ Dynamic weather icons based on current conditions
* 📱 Responsive design for desktop and mobile devices
* ⚡ Fast and seamless API integration

## Technologies Used

* HTML5
* CSS3
* JavaScript
* Weather API

This project demonstrates API integration, asynchronous JavaScript (Fetch API/AJAX), DOM manipulation, and responsive web design principles.
